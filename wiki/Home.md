# Dark Gravity: Autonomous Agent Factory

Welcome to the official wiki for **Dark Gravity**. This repository implements a **Continuous Agentic / Continuous Deployment (CA/CD)** autonomous factory following the **Onion Architecture**, **Spec-Driven Development (SDD)** with **GitHub Spec-Kit**, and **Domain-Driven Design (DDD)** standards.

The system orchestrates a specialized autonomous workforce — **Rustant** (PO), **ZeroClaw** (Developer), **DevOps Agent** (Self-Healing), and **Documentation Agent** (Memory) — through a durable 6-phase Hatchet DAG within a Zero Trust OpenZiti dark network.
